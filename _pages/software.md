Open-source tools for science policy research, developed in collaboration with [Andrew Stellman](http://www.stellman-greene.com/).

## PubHarvester

An open-source software tool designed to retrieve scientific publications data from MEDLINE.

\[[Website](http://www.stellman-greene.com/PublicationHarvester/)\]

## SC/Gen

An open-source software tool to create the coauthorship network for a set of focal scientists.

\[[Website](http://stellman-greene.com/SCGen/)\]

## Scientific Distance

An open-source software tool designed to calculate distance between any pair of scientists in "ideas space" based on the overlap of the MeSH keywords in their publications.

\[[Website](http://www.stellman-greene.com/ScientificDistance/)\]

## FindRelated

An open-source software tool designed to harvest efficiently the set of articles returned by the PubMed Related Citation algorithm, for any given set of PubMed identifiers (pmid's) provided as an input.

\[[Website](http://www.stellman-greene.com/FindRelated/)\]

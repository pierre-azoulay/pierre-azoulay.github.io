## Software

Open-source tools for science policy research, developed in collaboration with [Andrew Stellman](http://www.stellman-greene.com/).

**PubHarvester.** An [open-source software tool](http://www.stellman-greene.com/PublicationHarvester/) designed to retrieve scientific publications data from MEDLINE.

**FindRelated.** An [open-source software tool](http://www.stellman-greene.com/FindRelated/) designed to harvest efficiently the set of articles returned by the PubMed Related Citation algorithm, for any given set of PubMed identifiers (pmid's) provided as an input.
